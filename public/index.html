How to use it

  1. Open this page on two devices.
  2. Use the same room code on both devices, or send the room link.
  3. Allow microphone access.
  4. Press Start Call on one side.

For LAN testing with microphone access on another device, add local cert files and use HTTPS on port 3443.

Hosted version note: signaling is hosted, but audio is still peer-to-peer WebRTC. Some networks need a TURN relay or the call may connect with no audio.
